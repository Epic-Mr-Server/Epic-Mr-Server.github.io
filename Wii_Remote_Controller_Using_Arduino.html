Wii Remote Controller Using Arduino
 Date: June 17, 2021 → June 21, 2021
 Type: Machine Learning and Computer Science
 
Basic Explanation  

On a game controller, buttons are the go to method for interfacing with a game. A button is usually made of two disconnected wires that don’t activate anything until a bridge connects the two. For this project, I used an arduino board (a mini computer that can interface with electronic components) to bridge these two wires allowing me to interface with it with code.
